Brainwave Painting

The goal of this project is exploring the possibility of using brainwaves to interact with technology and art. The brainwave sensors will get the ratings of both attention and meditation of a person, and then change the colors of painting according to the rating level on screen in real time. The greater attention rate, the darker painting is (Maximum = 100, color = black), and vice versa. The brainwave sensor I used in this project is from NeuroSky and the program is coded in Processing.
